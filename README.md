# work
Shoes Shopping Mobile App – Low Fidelity Wireframe
Project Overview

This project is a low-fidelity wireframe design for a Shoes Shopping mobile application created in Figma. The design focuses on improving the product selection experience and making shoe browsing easier, more attractive, and user-friendly.

Objective

The goal of this wireframe is to create a simple shopping experience where users can quickly view shoe details and select products without unnecessary actions.

Screen Description
This screen represents the product selection state of the shopping application where users can:

View multiple shoe images
Check product information
Select shoe size
View ratings
Adjust quantity
Add products to basket
Buy directly using the "Shop Now" option

#Design Concept

The interface displays three product images on the left side.

One image is highlighted as the active selection
Other images remain visible but less emphasized
In the prototype, image transitions are animated
Users can quickly view all product options without repeatedly swiping

This approach was designed to:

Reduce extra user actions
Improve accessibility
Make product selection easier
Provide a visually attractive experience
Allow faster product comparison

#Design Thinking Process
Empathize
Users often find continuous swiping frustrating while browsing multiple product images.

Define
The problem identified was that users need a faster and simpler way to view multiple product options.

Ideate
A vertical image preview layout was considered where all shoe options remain visible.

Prototype
Created low-fidelity wireframes and interaction flow in Figma.

Test
The layout was reviewed to ensure users could easily view and select products.

Tools Used
Figma

#Key Features
Multiple product image previews
Highlighted product selection
Product details section
Quantity control
Add to Basket and Shop Now actions
Prototype animations

#Design Goal
To provide an easy, attractive, and efficient shopping experience with fewer interactions and faster product selection.
